## Installation

```bash
  npm install
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/Kusays/Ivashin-test.git
```

Go to the project directory

```bash
  cd Ivashin-test
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
